# Vk_desktop
Desktop App  for vk.com
----

1:
```sh
$ git clone git@github.com:prilipkoleg/vk_desktop.git .
```

2:
```sh
$ npm update
```

3:
```sh
$ bower update
```

4:
```sh
$ npm start
```


> Пример для Windows можно скачать по [ссылке].
> Чтобы открыть нужно запустить vk_desktop.exe.
